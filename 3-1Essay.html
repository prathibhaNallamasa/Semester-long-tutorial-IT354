Build a responsive website with HTML and CSS

In the video, he set the margins and padding to zero at the begining which was a great thing to do at the start as it messes up with the containers and the nav bars if you have any.
He also added the class to each nav link which makes it easy to differentiate from other links, if you want to style the links.
display: flex was something i did not know which is used to move the nav bar items in a line.
flex was used in different ways to arrange the nav bar items.

The linear-gradient was used to color the background with a shade which is darker at one side and fades towards the other side. he used properties like, To right and the colors of his choice to create that effect and he also used the -webkit-background-clip to clip the effect to the text and also -webkit-text-fill-color to fill the text with that effect.

The shadow was added to the image by applying filter: drop-shadow with appropriate arguments.

He created an animation using @keyframes to make the image drop slowly after loading the webpage.
