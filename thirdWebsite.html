Object creation

There are 4 different types of object creations. they are
Object creation with new keyword
object creation with literals
object creation with constructors
object creation with classes

Object creation with new keyword

In this type of object creation we create object with use of a new keyword
below is the example 

                        
                            var person = new Object()
                            person.firstName = "John";
                            person.lastName = "Doe";
                            person.age = 50;
                            person.firstName + " is " + person.age + " years old." // returns John is 50 years old.

object creation with literals

In this we do not use any keywords.

                    
                        var person = {
                            firstName = "John",
                           lastName = "Doe",
                           age = 50};
                            person.firstName + " is " + person.age + " years old." // returns John is 50 years old.

object creation with constructors

In this type of creation we use the constructors.You can see the following example

                    
                        function car(make,model,year)
                        {
                            this.make = make;
                            this.model = model;
                            this.year = year;
                        }
                        var mycar = new car('eagle','talon tsi',1993)
                        mycar.make // returns eagle

object creation with classes

In here we use classes and constructor for the class to create an object. 

                    
                        class Rectangle
                        {
                            constructor(height,width)
                            {
                                this.height = height;
                                this.width = width;
                            }
                            get area()
                            {
                                return this.calc();
                            }
                            calc()
                            {
                                return this.height * this.width;
                            }
                        }
                        const square = new Rectangle(20,20);
                        console.log(square.area) // returns 400
