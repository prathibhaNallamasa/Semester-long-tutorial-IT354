Node and V8 Engine

What is node?

Node.js is an open-source and cross platform javascript runtime environment which allows the users to run the JS code outside the web browser. Node.js uses the V8 engine(which is used in a chrome) outside the browser allowing the users to good performance.
Node.js was written initially by Ryan Dahl in 2009 which supports the creation of web servers and other tools using JS and the collection of modules which can handle the core functionalities. Node.js natively supported only JS, but now it can support compile-to-JS languages.


What is V8 engine

V8 is the javascript engine which powers the chrome. It is the open-source javascript engine which was developed by the Chromium project for the Google Chrome. It was created by Lars Bak and was released on September 2, 2008. V8 provides the environment for the chrome to run the javascript code while browsing on the chrome.
First V8 assembler was based on Strongtalk assembler but then in December 2010, the new infrastructure "Crankshaft" with speed improvemnets was released. TurboFan has been released with more speed in 2015 from Chrome.
In 2016, V8 was added with Ignition interpreter to reduce the memory usage on small memory of the Android phones. In 2017, a brand new compiler was released with V8.

How node uses V8

There's no compile task in node. V8 also uses a lot of different threads to make itself faster: The main thread is the one that fetches, compiles and executes JS code. Another thread is used for optimisation compiling so the main thread continues the execution while the former is optimising the running code
