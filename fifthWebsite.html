Modules

Modules are the set of functions you want to include in your application.

Modules can load each other and use special directives export and import to interchange functionality, call functions of one module from another one:
export keyword labels variables and functions that should be accessible from outside the current module.
import allows the import of functionality from other modules.

                        
                            // in index.js 

                            const circle = require('./circle');
                            const r = 3;
                          console.log(`Circle with radius ${r} has
                         area: ${circle.area(r)};
                         circumference: ${circle.circumference(r)}`);



  // in circle.js 
  
  const PI = 3.14;
exports.area = radius => (radius ** 2) * PI;
exports.circumference = radius => 2 * radius * PI;

Why use Modules?

Maintainability:

A well-designed module aims to lessen the dependencies on parts of the codebase as much as possible, so that it can grow and improve independently. Updating a single module is much easier when the module is decoupled from other pieces of code.

Namespacing:

In JavaScript, variables outside the scope of a top-level function are global. Because of this, it’s common to have “namespace pollution”, where completely unrelated code shares global variables. modules allow us to avoid namespace pollution by creating a private space for our variables.

Reusability:

The modules make the code easily available for the Reusability.



Higher order functions

A function that accepts and/or returns another function is called a higher-order function.
It’s higher-order because instead of strings, numbers, or booleans, it goes higher to operate on functions. Pretty meta.
With functions in JavaScript, you can:

  • Store them as variables
  • Use them in arrays
  • Assign them as object properties (methods)
  • Pass them as arguments
  • Return them from other functions

Below are two examples for higher-order functions:
Example 1: 

                            
                                //without higher order functions
                                console.log("example without higher order function")
                                const arr1 = [1, 2, 3];
                                const arr2 = [];
                                for(let i = 0; i < arr1.length; i++) {
                                 arr2.push(arr1[i] * 2);
                                    }
                                // prints [ 2, 4, 6 ]
                                console.log(arr2);

                                
                                // with higher order functions
                                console.log("example with higher order function")
                                const arr1 = [1, 2, 3];
                                const arr2 = arr1.map(function(item) {
                                  return item * 2;
                                });
                                console.log(arr2);
Example 2: 
                            
                                //without higher order functions
                                console.log("example without higher order function")
                                const birthYear = [1975, 1997, 2002, 1995, 1985];
                                const ages = [];
                                for(let i = 0; i < birthYear.length; i++) {
                                let age = 2018 - birthYear[i];
                                ages.push(age);
                                }
                                // prints [ 43, 21, 16, 23, 33 ]
                                console.log(ages);

                                //with higher order function
                                console.log("example with higher order function")
                                const birthYear = [1975, 1997, 2002, 1995, 1985];
                                const ages = birthYear.map(year => 2018 - year);
                                // prints [ 43, 21, 16, 23, 33 ]
                                console.log(ages);

                                    // pop() function
                                    console.log(" pop function");
                                    var fruits = ["Banana", "Orange", "Apple", "Mango"];

                                    console.log("after pop function");

                                    //push()

                                    let cats = ['Bob'];

                                    cats.push('Willy'); 

                                    cats.push('Puff', 'George');

Find() function:

The JavaScript Array.find method is a convenient way to find and return the first occurence of an element in an array, under a defined testing function. 

                                 
                                    //find()
                                    console.log("find function");
                                    const trees = [ 
                                      "birch", 
                                      "maple", 
                                      "oak", 
                                      "poplar" 
                                    ];
                                    
                                    const result = trees.find(tree => tree.startsWith("m"));
                                    result

filter() function:

filter() creates a new array with elements that fall under a given criteria from an existing array 

                                
                                    
                                            //filter()

                                            var numbers = [1, 3, 6, 8, 11];
                                            console.log("filter function");
                                            var lucky = numbers.filter(function(number) {
                                            return number > 7;
                                            });
                                            lucky

FindIndex() method:

The findIndex() method returns the index of the element that satisfies a testing function or -1 if no element passed the test. 

                                
                                    //findIndex()
                                    console.log("findIndex function");
                                    let ranks = [1, 5, 7, 8, 10, 7];
                                    let index = ranks.findIndex(rank => rank === 7);
                                    console.log(index);

ForEach() method:

JavaScript Array provides the forEach() method that allows you to run a function on every element. 

                                
                                    //forEach()
                                    console.log("forEach function");
                                    let ranks = ['A', 'B', 'C'];

                                    ranks.forEach(function (e) {
                                        console.log(e);
                                    });
